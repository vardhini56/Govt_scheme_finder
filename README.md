# Govt_scheme_finder
This project helps in finding the government schemes within a fraction of seconds using AI knowledge

A web application that helps Andhra Pradesh citizens easily discover eligible government schemes. The system provides AI-based recommendations, filtering schemes based on user details like:

Age
Gender
Income
Occupation
Education
Category (Student / Farmer / Women / Senior citizen)
Key Features
Smart Scheme Recommendation 🤖
User Eligibility Checker
Search & Filter Schemes
Responsive UI
Backend API Integration
Database for Scheme Data
